# Temperature Converter

## Description
This project is a responsive temperature converter built using HTML5, CSS3, and JavaScript. It converts temperatures between Celsius, Fahrenheit, and Kelvin with instant results.

## Features
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Kelvin Conversion
- Input Validation
- Responsive Design
- User Friendly Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your browser.
